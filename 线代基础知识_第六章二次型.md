## 二、基本内容与重要结论

### 基础知识

**定义 6.1** 含有 n 个变量 x₁, x₂, …, xₙ 的二次齐次函数
f(x₁, x₂, …, xₙ) = a₁₁x₁² + a₁₂x₁x₂ + … + a₁ₙx₁xₙ
                  + a₂₁x₂x₁ + … + a₂ₙx₂xₙ
                  ⋮
                  + aₙ₁xₙx₁ + … + aₙₙxₙ²

称为 n 元二次型，若规定 aᵢⱼ = aⱼᵢ, ∀i, j = 1, 2, …, n，则二次型有矩阵表示
f(x₁, x₂, …, xₙ) = x^T Ax

其中 x = (x₁, x₂, …, xₙ)^T, A = [aᵢⱼ] 且 A^T = A 是对称矩阵，称 A 为二次型
的矩阵，秩 r(A) 称为二次型的秩。

例如：二元二次型 f(x₁, x₂) = x₁² + 5x₁x₂ + 6x₂, x₁
f(x₁, x₂) = x₁² + 3x₁x₂ + 3x₂x₁ + 5x₂²
         = [x₁, x₂]$\begin{bmatrix} 1 & 3 \\ 3 & 5 \end{bmatrix}$$\begin{bmatrix} x_1 \\ x_2 \end{bmatrix}$
         = x^T Ax

为二次型的矩阵表示。

**定义 6.2** 如果二次型中只含有变量的平方项，所有混合项 xᵢxⱼ（i ≠ j）的系数全为零，即
$x^TAx = d_1x_1^2 + d_2x_2^2 + \cdots + d_nx_n^2$

这样的二次型称为标准形。
在标准形中，若系数 dᵢ 为 1, -1 或 0，即
$x^TAx = x_1^2 + x_2^2 + \cdots + x_r^2 - x_{r+1}^2 - \cdots - x_{r+s}^2$

的形式，称为二次型的规范形。

**定义 6.3** 在二次型 x^T Ax 的标准形中，正平方项的个数 p 称为二次型的正惯性指数，负平方项的个数 q 称为二次型的负惯性指数。

**定义 6.4** 如果
$\begin{cases}
r_1 = c_{11}y_1 + c_{12}y_2 + c_{13}y_3\\
r_2 = c_{21}y_1 + c_{22}y_2 + c_{23}y_3\\
r_3 = c_{31}y_1 + c_{32}y_2 + c_{33}y_3
\end{cases}$

满足
$|C| = \begin{vmatrix} c_{11} & c_{12} & c_{13} \\ c_{21} & c_{22} & c_{23} \\ c_{31} & c_{32} & c_{33} \end{vmatrix} \neq 0$

称（6.4）为由 x = (x₁, x₂, x₃)^T 到 y = (y₁, y₂, y₃)^T 的坐标变换。

【注】坐标变换（6.4）用矩阵表示，即
$\begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix} = \begin{bmatrix} c_{11} & c_{12} & c_{13} \\ c_{21} & c_{22} & c_{23} \\ c_{31} & c_{32} & c_{33} \end{bmatrix} \begin{bmatrix} y_1 \\ y_2 \\ y_3 \end{bmatrix}$ 或 $x = Cy$

其中 C 是可逆矩阵。

**定义 6.5** 两个 n 阶矩阵 A 和 B，如果存在可逆矩阵 C，使得
$C^TAC = B$（6.5）

就称矩阵 A 和 B 合同，记作 A～B。将矩阵 A 到 B 的变换为合同变换，称 C 为合同变换的矩阵。

如 A～B, B～C，则 A～C
因为 P^T AP = B, P 可逆，Q^T BQ = C, Q 可逆，
于是 (PQ)^T A(PQ) = Q^T(P^T AP)Q = Q^T BQ = C，
有 P = P, Q 可逆且 P^T AP = C。

**定义 6.6** 对二次型 x^T Ax，如果对任何 x ≠ 0，有 x^T Ax > 0，则称二次型 x^T Ax 是正定二次型，对称矩阵 A 是正定矩阵。

例如：二次型 f(x₁, x₂, x₃) = x₁² + 5x₂² - 4x₁x₃ + 2x₁x₂x₃ 的矩阵
数是 -4，如果取 x = (0, 0, 1)^T ≠ 0，则有
$f(0, 0, 1) = 0$，此时 A 不是正定矩阵。

设明这个二次型不是正定的。二次型的矩阵
$A = \begin{bmatrix} 1 & 1 & -2 \\ 1 & 5 & 0 \\ -2 & 0 & -1 \end{bmatrix}$

也不是正定矩阵。

变换坐标考虑 f(x₁, x₂, x₃) = x₁² + 5x₂² + 2x₁x₂x₃，取 x = (0, 0, 1)^T ≠ 0，有 f(0, 0, 1) = 0，此知 A 正定的充要条件是 xᵢ > 0(i = 1, 2, 3)。

如何判断 $\begin{vmatrix} 1 & 2 & 1 \\ 2 & 5 & 2 \\ 1 & 2 & 6 \end{vmatrix}$, $\begin{vmatrix} 2 & 3 & 2 \\ 3 & 5 & 2 \\ 2 & 2 & 3 \end{vmatrix}$, $\begin{vmatrix} -1 & 2 & 1 \\ 2 & -1 & 1 \\ 1 & 1 & 6 \end{vmatrix}$ 正定？

### 重要结论

**定理 6.1** 变量 x = (x₁, x₂, …, xₙ)^T 的 n 元二次型 x^T Ax 经坐标变换 x = Cy，化为变量 y = (y₁, y₂, …, yₙ)^T 的 n 元二次型 y^T By，其中 B = C^T AC。

若用 x^T Ax = (Cy)^T A(Cy) = y^T C^T ACy = y^T By 表示坐标变换 x = Cy 后二次型的矩阵表示，则 y₁, y₂, …, yₙ 为自变量的二次型表达式。

标变换 x = Cy 化为 y₁, y₂, …, yₙ 为自变量的二次型，二次型矩阵是合同的。

特别地，若 x = Cy 且 C 是正交矩阵，则有
$B = C^TAC = C^TAC = C^TAC$

因经过正交变换，二次型矩阵不仅合同且相似。

**定理 6.2** 任意的 n 元二次型 f(x₁, x₂, …, xₙ) = x^T Ax 经过适当的可逆线性变换可以化为标准形
$d_1y_1^2 + d_2y_2^2 + \cdots + d_ny_n^2$

其中 dᵢ (i = 1, 2, …, n) 是常数。

**定理 6.3** 任一 n 阶实对称矩阵 A，总可以合同于一个对角矩阵，即
$C^TAC = \begin{bmatrix} d_1 & & \\ & \ddots & \\ & & d_n \end{bmatrix}$（6.6）

**定理 6.4**（惯性定理）对于一个二次型，不论是通过什么样的标准坐标变换化为没有交叉项的标准形，其中正平方项的个数 p，负平方项的个数 q，而平方项的下标项的个数 n - p - q 是二次型矩阵一确定的。

**定理 6.5** 对任一个 n 元二次型 x^T Ax，其中 A 是 n 阶实对称矩阵，必在正交变换 x = Qy（Q 是正交矩阵）下使得 x^T Ax 化为标准形
$x^TAx = \lambda_1y_1^2 + \lambda_2y_2^2 + \cdots + \lambda_ny_n^2$

其中 λ₁, λ₂, …, λₙ 是 A 的 n 个特征值。

【注意】A 为实对称矩阵，且 A 的特征值 λᵢ 互不相同，则 A 的特征向量 Q = [γ₁, γ₂, …, γₙ] 是正交矩阵，使得 x^T Ax 化为标准形
$x^TAx = \lambda_1y_1^2 + \lambda_2y_2^2 + \cdots + \lambda_ny_n^2$

这里 λᵢ, λ₂, …, λₙ 是 A 的 n 个特征值。

**定理 6.6** n 元二次型 x^T Ax 正定的充分必要条件是：
（1）A 的顺序主子式数 Δᵢ > 0；
（2）A 的所有特征值 λ(i = 1, 2, …, n) 均为正数；
（3）A 的所有特征值 λ(i = 1, 2, …, n) 均为正数；
（4）A 的合同标准形正交均为正的平方项；
满足 x^T Ax 正定的充要条件是：
（1）dᵢ > 0(i = 1, 2, …, n)；
（2）|A| > 0。

基础变换（1）配方法（2）正交变换法。

## 正交变换化标准形

提示：$x^TAx \xrightarrow{x=Qy} \lambda_1y_1^2 + \lambda_2y_2^2 + \lambda_3y_3^2$
$Q = [γ_1, γ_2, γ_3]$ 特征向量

【评注】本节提用正交变换化二次型为标准形的方法，标准形中平方项的系数是二次型矩阵的特征值，所用的正交变换矩阵是由特征向量组成的矩阵 P = [γ₁, γ₂, …, γₙ]
（1）写出二次型矩阵 A；
（2）求特征方程 |λI - A| = 0；
（3）求矩阵 A 的特征向量 γ₁, γ₂, …, γₙ；
（4）求规范特征向量（单位化，Schmidt 正交化）γ₁, γ₂, γ₃；
（5）构造正交变换 P = [γ₁, γ₂, γ₃]。
则标准变换 x = Py 得
$x^TAx = \lambda_1y_1^2 + \lambda_2y_2^2 + \lambda_3y_3^2$

【注意】特征值的顺序与正交变换 P 中对应的特征向量的顺序是一致的。

## 规范形

提示：$x_1^2 + \cdots + x_p^2 - x_{p+1}^2 - \cdots - x_{p+q}^2$

## 二次型的正定性

提示：∀x ≠ 0, x^T Ax > 0
特征值 λᵢ 均为正，p = n，A～C^T C，C 可逆。

## 矩阵的等价、相似、合同

### 1. A 和 B 等价⟺秩相同，同阶矩阵
A 与 B 等价⟺A 经初等变换变为 B
⟺P A Q = B，其中 P, Q 可逆
⟺r(A) = r(B)。

### 2. A 和 B 相似⟺特征相同
A 与 B 相似⟺∃ 可逆矩阵 P，使 P⁻¹AP = B。
判断：证明相似
（1）A - λI 与 B - λI 相似；
（2）实对称矩阵 A～B⟺λᵢ = λᵢ。
判断不相似：
（1）∑λᵢ ≠ ∑λᵢ 或 λᵢ ≠ λᵢ(i) 或 |A| ≠ |B| 或 A ≠ B；
（2）A 可相似对角化，B 不能相似对角化。

### 3. A 和 B 合同⟺惯性指数相同
A 与 B 合同⟺C^T AC = B，其中 C 可逆。
⟺x^T Ax 与 y^T By 有相同的正、负惯性指数。

【例题 2】 矩阵 $A = \begin{bmatrix} 0 & 0 \\ 0 & 1 \end{bmatrix}$，$B = \begin{bmatrix} 1 & 0 \\ 0 & 0 \end{bmatrix}$ 等价，合同不相似。

解：因为 r(A) = r(B)，所以 A 与 B 等价。
