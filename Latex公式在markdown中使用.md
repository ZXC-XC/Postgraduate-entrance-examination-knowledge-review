# 大括号函数
```
$$
\operatorname{sgn}(x) =           % 左边的符号函数名称
\begin{cases}                     % 开始分段函数格式
-1, & x < 0 \\                    % 第一种情况：x 小于 0 时函数为 -1
0, & x = 0 \\                     % 第二种情况：x 等于 0 时函数为 0
1, & x > 0                        % 第三种情况：x 大于 0 时函数为 1
\end{cases}                       % 结束分段函数
$$
```
